Authoring Tool

Welcome, You can use this tool to create new content or modify existing ones.

  • Use “Create” to start creating new content. Once the content is saved, it would be available in Draft.
  • To Edit content in “Sent For Review”, it needs to be recalled before you start editing.
  • "Published" content can be edited from the Published section. Once you start editing, the content will be available in the Draft section.
